# Toast, Alert & Dialog Service Plan

## Overview

A centralized notification and dialog system for the Blazor WebAssembly application.

---

## Components

| Component | Purpose | Behavior |
|-----------|---------|----------|
| **Toast** | Brief feedback messages | Auto-dismiss (3-5s), stackable, corner position |
| **Confirmation Dialog** | Get user consent | Modal, blocks until answered, returns `bool` |
| **Alert Dialog** | Important acknowledgment | Modal, single OK button |
| **Prompt Dialog** | Get text input | Modal, text field + Confirm/Cancel, returns `string?` |

## Service Interfaces

### IToastService.cs

```csharp
public interface IToastService
{
    void Success(string message, string? title = null, int durationMs = 5000);
    void Error(string message, string? title = null, int durationMs = 5000);
    void Warning(string message, string? title = null, int durationMs = 5000);
    void Info(string message, string? title = null, int durationMs = 5000);
    
    event Action<ToastMessage>? OnToastAdded;
    event Action<Guid>? OnToastRemoved;
}
```

### IDialogService.cs

```csharp
public interface IDialogService
{
    Task<bool> ConfirmAsync(string message, string? title = null, 
        string confirmText = "Confirm", string cancelText = "Cancel",
        DialogVariant variant = DialogVariant.Default);
    
    Task AlertAsync(string message, string? title = null, 
        string okText = "OK");
    
    Task<string?> PromptAsync(string message, string? title = null,
        string? defaultValue = null, string? placeholder = null,
        string confirmText = "OK", string cancelText = "Cancel");
}
```

## Usage Examples

### Toasts (Fire and Forget)

```csharp
@inject IToastService Toast

// Success
Toast.Success("Settings saved successfully!");

// Error
Toast.Error("Failed to delete item.");

// Warning
Toast.Warning("Your session expires in 5 minutes.");

// Info
Toast.Info("Processing your request...");

// Custom duration
Toast.Success("Done!", durationMs: 3000);
```

### Confirmation Dialog (Await Result)

```csharp
@inject IDialogService Dialog

// Basic confirmation
if (await Dialog.ConfirmAsync("Delete this item?"))
{
    await DeleteItemAsync();
}

// With custom options
if (await Dialog.ConfirmAsync(
    "Are you sure you want to delete this item? This cannot be undone.",
    title: "Delete Item",
    confirmText: "Delete",
    cancelText: "Keep",
    variant: DialogVariant.Danger))
{
    await DeleteItemAsync();
}
```

### Alert Dialog (Acknowledgment)

```csharp
@inject IDialogService Dialog

await Dialog.AlertAsync("Your session has expired.", "Session Timeout");
```

### Prompt Dialog (Get Input)

```csharp
@inject IDialogService Dialog

var newName = await Dialog.PromptAsync(
    "Enter a new name for the file:",
    title: "Rename File",
    defaultValue: currentName,
    placeholder: "File name");

if (newName != null)
{
    await RenameFileAsync(newName);
}
```

## UI Design Notes

### Toast Container
- **Mobile (< md)**: Top-center
- **Desktop (â‰¥ md)**: Top-right
- Stack direction: Newest on top
- Animation: Slide in, fade out on dismiss
- Max visible: 5 (oldest auto-dismissed if exceeded)

**Responsive Tailwind classes:**
```html
<div class="fixed top-4 left-1/2 -translate-x-1/2 md:left-auto md:right-4 md:translate-x-0 z-50">
```

### Dialog Container
- Centered modal with backdrop
- Backdrop click = cancel (for confirm/prompt)
- Focus trapped inside dialog
- Escape key = cancel

### Styling (Tailwind)
- Success: `bg-green-100 border-green-500 text-green-800 dark:bg-green-900 dark:border-green-700 dark:text-green-200`
- Error: `bg-red-100 border-red-500 text-red-800 dark:bg-red-900 dark:border-red-700 dark:text-red-200`
- Warning: `bg-amber-100 border-amber-500 text-amber-800 dark:bg-amber-900 dark:border-amber-700 dark:text-amber-200`
- Info: `bg-blue-100 border-blue-500 text-blue-800 dark:bg-blue-900 dark:border-blue-700 dark:text-blue-200`
